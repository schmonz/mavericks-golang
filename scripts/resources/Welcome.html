This installs a modern Go 1.26 toolchain, patched to build and run on Mac OS X 10.9 (Mavericks), into /usr/local/go126/.

Add /usr/local/go126/bin to your PATH, then go build works out of the box — cgo included — producing 10.9-safe binaries with modern TLS (verified against the system keychain plus a bundled CA set).

A background updater (Sparkle) checks daily for new releases. Built from go.dev source; no Apple prebuilt toolchain is redistributed.
